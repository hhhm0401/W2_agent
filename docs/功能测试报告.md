## 4. 后台精准计时验证方法

核心算法的确定性测试已覆盖 `endAt - now` 计算。人工复测步骤：

1. 开始一轮专注并记下显示值。
2. 将页面置于后台 60–120 秒。
3. 切回页面，触发 `visibilitychange` 和 `focus` 校准。
4. 与系统时钟差比较，允许显示向上取整导致小于 1 秒的视觉差。
5. 刷新页面，确认恢复后的剩余时间继续基于保存的 `endAt`。

本次自动环境未用固定等待占用 120 秒；最终报告不虚构未经人工长时等待测得的误差数字。
